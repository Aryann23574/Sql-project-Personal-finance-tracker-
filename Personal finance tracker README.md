# Personal Finance Tracker

## Introduction

The Personal Finance Tracker is a SQL-based database project used to manage personal income and expenses. It helps users monitor spending and calculate their remaining balance.

## Features

* Store user information
* Record income
* Record expenses
* Categorize expenses
* Generate spending reports
* Calculate available balance

## Tools Used

* MySQL
* OneCompiler (MySQL)

## Database Tables

* Users
* Categories
* Income
* Expenses

## SQL Concepts Used

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT
* SELECT
* JOIN
* SUM()
* GROUP BY

## Conclusion

This project demonstrates how SQL can be used to manage financial data and generate useful reports. It provides a simple and effective way to track income, expenses, and overall balance.
