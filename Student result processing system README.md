# Student Result Processing System

## Introduction

The Student Result Processing System is a SQL-based database project designed to store and manage student academic records. It keeps track of students, courses, and marks, and generates reports such as average marks and student rankings.

## Features

* Store student details
* Store course information
* Record student marks
* Calculate average marks
* Generate student rank list using SQL

## Tools Used

* MySQL
* OneCompiler (MySQL)

## Database Tables

* Students
* Courses
* Grades

## SQL Concepts Used

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT
* SELECT
* JOIN
* GROUP BY
* AVG()
* RANK() Window Function

## Conclusion

This project demonstrates the basic concepts of SQL database design and data analysis. It is useful for understanding relationships between tables and generating reports using SQL queries.
